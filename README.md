# Proyecto evaluacion - Frontend

Este repositorio es un proyecto de práctica utilizado para implementar conceptos de desarrollo web, pruebas de maquetación y configuraciones de despliegue en entornos de producción.

**🔗 Enlace al proyecto:** [Visitar sitio en producción](https://enmanuel-miranda.github.io/proyecto_ejemplo.github.io/)

---

## ⚙️ Funcionalidades clave

- **Prueba de Despliegue:** Configuración y sincronización exitosa con GitHub Pages.
- **Estructura Base:** Maquetación limpia ideal para testear estilos y componentes visuales.
- **Diseño Adaptativo:** Ajustes básicos para comprobar la visualización en entornos móviles.

## 🛠️ Tecnologías utilizadas

- **HTML5** / **CSS3**
- **JavaScript**
- **GitHub Pages** (Despliegue)
